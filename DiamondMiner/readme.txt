Resumo:
O jogo é o baseado no jogo Diamond Miner, mas versão natalícia, no qual o jogador precisa de apanhar os  presentes em cada um dos três níveis, para ganhar o jogo. Ao longo do jogo, o jogador parte a terra coebrta de neve que se encontra pelo caminho, assim como certas paredes de gelo, mas tem de ter cuidado para que nenhum rochedo lhe caia em cima, senão perde uma das suas três vidas. Caso o jogador se engane, existe a possibilidade de dar reset ao nível, mas ao fazer isto perde uma vida.

Comandos:
- WASD para o jogador se mover;
- E para usar a dinamite;
- R para dar reset.
